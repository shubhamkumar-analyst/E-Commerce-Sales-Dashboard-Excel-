# E-Commerce-Sales-Dashboard-Excel-
A sample dataset used for building the dashboard is included in this repository.
## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Analysis
- Power Query

This project is an interactive E-Commerce Sales Dashboard built in Microsoft Excel to analyze sales performance, customer orders, and delivery efficiency. The dashboard helps identify key business insights such as revenue trends, top performing states and cities, and delivery performance.
